About the tool

The original idea of making patters with CSS gradient function came from a presentation by Lea Verou and her CSS patterns gallery. We found this very intresting and wanted to make a tool that allow web developers make patterns more easy. There is endless posibilities for creating creative patterns using CSS. We are trying to make the tool as flexable as we can to give developers more posibilites.

Browser Support

This tool is for generating code for modern websites. So we just support modern browsers. With that we chose not to even try to support older versions of IE! CSS background-size propery is something new (As now Nov 2011) and not even all modern browsers supporting that properly. Caniuse.com have an up to date browser support table for that. background-image:-prefix-linear-gradient() have wider browser support. But background-image:-prefix-radial-gradient() supported in less browsers.

We always test generated code and update this browser support table:
